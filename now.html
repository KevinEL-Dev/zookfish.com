So in my last update I found a command that I thought provided the total cpu% utilization under the parent process of hytale.

Below is the command:

top -b -d 1 -n 2 | awk '$1 == "PID" {block_num++; next} block_num == 2 {sum += $9;} END {print sum}'

So when I run the command above it only retrieves the one hytale process and its not doing what I expected it to do.
